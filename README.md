JQuery Plugin Hello World Demo
==============================

JQuery Plugin，就是在JQuery上提供一些自己的功能，这些功能和JQuery自己的函数融为一体，看起来就像是JQuery自己的。

JQuery Plugin写起来非常简单，很好。

```
npm install
open index.html
```
